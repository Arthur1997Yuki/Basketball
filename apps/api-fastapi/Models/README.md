# Models

ドメインモデル（エンティティ/値オブジェクト/リポジトリIF）をまとめた層です。

## 構成

- `Teams/` チーム関連
- `HomeTowns/` 本拠地関連
- `TeamAffiliations/` 所属関連
- `Seasons/` シーズン関連
- `Players/` 選手関連
- `Plays/` プレー関連
- `PlayEventLists/` プレーイベント一覧
- `Events/` イベント
- `OccurrenceTimings/` 発生タイミング
- `Areas/` エリア種別
- `Results/` 結果/フラグ
- `Tags/` タグ
- `MatchVideos/` 試合動画関連
