# Glossary

共通の用語集です。各 README から参照します。

## 一般

- バックエンド: API や DB へのアクセスを担当するサーバー側。
- フロントエンド: 画面表示とユーザー操作を担当するクライアント側。

## API / バックエンド

- FastAPI: Python 製の Web API フレームワーク。
- OpenAPI: REST API の仕様を記述する標準フォーマット。
- ORM: オブジェクトと DB テーブルを対応づける仕組み。
- UoW（Unit of Work）: 複数の DB 操作を 1 つのトランザクションとして扱う設計パターン。

## フロントエンド

- Next.js: React ベースのフルスタックフレームワーク。
- App Router: `app/` ディレクトリでルーティングを定義する仕組み。
- CSR/SSR: クライアント/サーバーでレンダリングする方式。

## DB / マイグレーション

- マイグレーション: DB スキーマ変更を履歴として管理する仕組み。
- リビジョン: マイグレーションの単位（1 回の変更）。

## ドキュメント

- PlantUML: テキストから図を生成するツール。
- `*.pu`: PlantUML のソースファイル拡張子。
